## 📌 معرفی

**NEX-GDDP-CMIP6 Downloader** یک نرم‌افزار دسکتاپ ویندوزی است که فرآیند دریافت داده‌های اقلیمی با وضوح بالا از سرورهای NASA NEX-GDDP-CMIP6 را از طریق یک رابط گرافیکی ساده و کاربرپسند آسان می‌کند.

این مجموعه داده شامل متغیرهای اقلیمی روزانه با **تفکیک مکانی ۰٫۲۵ درجه** است، شامل:

- داده‌های اقلیمی تاریخی
- پیش‌بینی‌های اقلیمی آینده تحت سناریوهای مختلف انتشار:

  - SSP1-2.6
  - SSP2-4.5
  - SSP3-7.0
  - SSP5-8.5

# 🚀 شروع کار

## روش اول — نسخه اجرایی ویندوز (پیشنهادی)

در این روش نیازی به نصب Python ندارید.

1. وارد بخش **Releases** شوید.
2. آخرین نسخه را دانلود کنید:

```
NEX-GDDP-CMIP6-Downloader.exe
```

3. برنامه را اجرا کنید.

---

## روش دوم — اجرای سورس پایتون

### پیش‌نیازها

- Windows 10 / 11
- Python نسخه 3.7 یا بالاتر
- pip

### نصب

ابتدا مخزن را دریافت کنید:

```bash
git clone https://github.com/haadiasghari/nex-gddp-cmip6-downloader.git

cd nex-gddp-cmip6-downloader
```

کتابخانه‌های مورد نیاز را نصب کنید:

```bash
pip install requests
```

### اجرای برنامه

```bash
python downloader.py
```

## 📂 ساختار خروجی

فایل‌های دانلودشده به صورت خودکار مرتب می‌شوند:

```text
└── ACCESS-CM2/

        └── r1i1p1f1/

                ├── tasmax_day_ACCESS-CM2_ssp245_r1i1p1f1_gn_2016_v2.0.nc
                └── ...
```

## ⚙️ تنظیمات برنامه

| گزینه | مقدار پیش‌فرض | توضیح |
|--------|---------------|-------|
| مسیر خروجی | `./cmip6_downloads` | محل ذخیره فایل‌ها |
