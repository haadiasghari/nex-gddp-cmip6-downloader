## 📌 معرفی

**NEX-GDDP-CMIP6 Downloader** یک برنامه دسکتاپ برای ویندوز است که دانلود داده‌های اقلیمی با وضوح بالا از مجموعه داده NASA NEX-GDDP-CMIP6 را ساده می‌کند.

این برنامه دریافت داده‌های اقلیمی روزانه با تفکیک مکانی **۰٫۲۵ درجه** را از سرورهای NASA آسان می‌کند.

این داده‌ها شامل:

- داده‌های اقلیمی دوره تاریخی
- پیش‌بینی‌های اقلیمی آینده بر اساس سناریوهای مختلف انتشار:

  - SSP1-2.6
  - SSP2-4.5
  - SSP3-7.0
  - SSP5-8.5

# 🚀 شروع استفاده

## روش اول — نسخه آماده ویندوز (پیشنهادی)

در این روش نیازی به نصب Python ندارید.

1. وارد بخش **Releases** شوید.
2. آخرین نسخه برنامه را دانلود کنید:

```
NEX-GDDP-CMIP6-Downloader.exe
```

3. فایل را اجرا کنید.

---

## روش دوم — اجرای برنامه از روی سورس پایتون

### نیازمندی‌ها

- Windows 10 یا Windows 11
- Python نسخه 3.7 یا بالاتر
- pip

### دریافت پروژه

```bash
git clone https://github.com/haadiasghari/nex-gddp-cmip6-downloader.git

cd nex-gddp-cmip6-downloader
```

### نصب کتابخانه‌ها

```bash
pip install requests
```

### اجرای برنامه

```bash
python downloader.py
```

## 🗂️ نحوه کار با برنامه

فرآیند دانلود در ۵ مرحله انجام می‌شود:

| مرحله | توضیح |
|---|---|
| 1️⃣ | انتخاب مدل اقلیمی CMIP6 |
| 2️⃣ | انتخاب سناریوی اقلیمی |
| 3️⃣ | انتخاب اعضای مدل (Ensemble) |
| 4️⃣ | انتخاب متغیرهای اقلیمی |
| 5️⃣ | تنظیم دانلود و شروع دریافت فایل‌ها |

## 📂 ساختار فایل‌های خروجی

فایل‌های دانلودشده به صورت خودکار مرتب می‌شوند:

```text
cmip6_downloads/
└── ACCESS-CM2/
    └── ssp245/
        └── r1i1p1f1/
            └── tasmax/
                ├── tasmax_day_ACCESS-CM2_ssp245_r1i1p1f1_gn_2015_v2.0.nc
                ├── tasmax_day_ACCESS-CM2_ssp245_r1i1p1f1_gn_2016_v2.0.nc
                └── ...
```

## ⚙️ تنظیمات دانلود

| گزینه | مقدار پیش‌فرض | توضیح |
|---|---|---|
| پوشه خروجی | `./cmip6_downloads` | محل ذخیره فایل‌ها |
